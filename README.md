## UI Design Principle

**Disappear.** The interface succeeds when you forget it exists.

The user's attention is sacred. Content is the product—everything else serves it. Controls reveal on intent, hide when not needed. State is transparent when visible, silent when not. The best feature is the one you don't notice working.
